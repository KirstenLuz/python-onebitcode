# Módulo 3 – Módulos e Bibliotecas  
(os, webbrowser, math, statistics, re, hashlib, json, collections, random, tkinter, venv)

Objetivo: Usar bibliotecas padrão e externas em aplicações práticas.  

1. Crie e importe um módulo próprio com uma função simples.  
2. Use `os.getcwd()` para exibir o diretório atual.  
3. Liste todos os arquivos e pastas do diretório atual.  
4. Abra um site no navegador usando `webbrowser`.  
5. Calcule a raiz quadrada de 144 com `math.sqrt()`.  
6. Use `math.factorial()` para calcular o fatorial de 6.  
7. Calcule a média de `[10, 5, 8]` usando `statistics.mean()`.  
8. Encontre todas as letras entre “a” e “m” em uma string usando regex.  
9. Criptografe uma senha usando `hashlib.sha256()`.  
10. Converta um dicionário Python para JSON e exiba a string.  
11. Leia um arquivo `.json` e exiba apenas uma chave específica.  
12. Use `Counter` para contar itens repetidos em uma lista.  
13. Use `namedtuple` para criar e armazenar dados de um jogo.  
14. Use `deque` para adicionar e remover elementos nas duas extremidades.  
15. Gere um número aleatório entre 1 e 100.  
16. Escolha aleatoriamente um elemento de uma lista.  
17. Embaralhe uma lista de cartas usando `random.sample()`.  
18. Crie uma janela simples com `tkinter` com um campo de texto e botão.  
19. Crie e ative um ambiente virtual (comandos no terminal).  
20. Gere um `requirements.txt` com `pip freeze`.
